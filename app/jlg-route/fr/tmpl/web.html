Développement web spécifique

Notre équipe est spécialisée dans les 3 « stacks » suivantes : PHP, NodeJS/Javascript, Java.

Nous travaillons souvent avec les framework suivants :

  • AngularJS/Angular
  • React
  • Vue
  • Polymer
  • Etc.

Nos sites sont responsives c'est-à-dire tout autant accessibles sur ordinateur de bureau que smartphone ou tablette.

Nous pouvons aussi nous occuper de développer des interfaces back-office de type REST API et autres formes de webservices avec des technologies telles que Java (JavaEE, Spring), NodeJS (Express), ou PHP (Zend, Symphony).

Nous pouvons aussi développer des écrans de visualisation dynamiques de données 2D ou 3D avec :

  • D3
  • ThreeJS

Maintenance de site web

Nous effectuons cette mission par gestion de tickets.

Quand vous désirez faire une modification sur votre site web, vous nous la décrivez, la faites parvenir par mail ou par un système de tickets comme Trello, JIRA, etc.

  • Ensuite, nous évaluons et vous communiquons le nombre de tickets nécessaires pour réaliser la modification. Nous garantissons une réponse sous 1 jour ouvré.
  • Vous envoyé votre « go » ou votre « no-go ».
  • Dans le cas d'un « go », nous réalisons la mission et ses tests afférents (Dev, Test, UAT, Prod).
  • Etc.

Nous pouvons vous aidez à maintenir votre site internet pour les technologies suivantes :

  • Principaux CMS: Wordpress, Drupal, etc.
  • Principaux eCommerce: WooCommerce, Prestashop, etc.
  • Sites spécifiques (Angular, React, Vue, etc.)

Tarif

Un montant forfaitaire mensuel additionné des tickets traités.

Le montant forfaitaire et le prix unitaire du ticket sont estimés en fonction de la complexité du site web, le niveau de disponibilité demandée et le volume.
